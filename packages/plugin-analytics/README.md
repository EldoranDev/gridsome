# @gridsome/plugin-analytics

> Adds Google Analytics to the Page

## Install
- `yarn add @gridsome/plugin-analytics`
- `npm install @gridsome/plugin-analytics`

## Usage

```js
module.exports = {
  plugins: [
    {
      use: '@gridsome/plugin-analytics',
      options: {
        id: 'UA-XXX-XX',
      }
    }
  ]
}
```
